# hello-world

Demo CMDOP skill — greets users by name.

When the user asks to greet someone, run `run.py` with the `greet` command.
When the user asks to say goodbye, run `run.py` with the `goodbye` command.

## Usage

```
cmdop hello-world "say hello to Alice"
```

## Commands

- `greet --name <NAME> [--shout]` — say hello (optionally in UPPERCASE)
- `goodbye --name <NAME>` — say goodbye
